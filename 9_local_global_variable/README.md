# 📘 PHP Learning – Day 9

## 📌 Topic: Local & Global Variables

Today I learned about **Variable Scope in PHP** and how variables behave inside and outside functions.

---

# 🔹 What is Variable Scope?

Variable scope defines **where a variable can be accessed in the program**.

In PHP, variables can have different scopes depending on where they are declared.

---

# 🔹 Local Variable

A **local variable** is declared inside a function and can only be used within that function.

---

# 🔹 Global Variable

A **global variable** is declared outside any function and can be accessed throughout the script.
