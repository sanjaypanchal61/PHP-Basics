# 📘 PHP Learning – Day 5

## 📌 Topic: Loops , Break and Continue

Today I learned different types of loops in PHP and how break and continue work.

---

# 🔹 Types of Loops

- for loop
- while loop
- do...while loop

---

# 🔹 break Keyword

The `break` statement immediately stops the loop execution.

---

# 🔹 continue Keyword

The `continue` statement skips the current iteration and moves to the next one.
