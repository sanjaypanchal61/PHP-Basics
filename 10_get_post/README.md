# 📘 PHP Learning – Day 10

## 📌 Topic: GET & POST

Today I learned how to send and receive form data using GET and POST methods.

---

# 🔹 GET vs POST

| Feature | GET | POST |
|------|------|------|
| Data Visible | Yes | No |
| Security | Low | Better |

---

# 🔹 $_GET

`$_GET` is a superglobal array used to collect form data sent through URL.

---

# 🔹 $_POST

`$_POST` is a superglobal array used to collect form data sent through HTTP request body.
