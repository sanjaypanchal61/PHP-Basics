# 📘 PHP Learning – Day 11

## 📌 Topic: Include & Require

Today I learned how to reuse code using include and require in PHP.

---

# 🔹 include

include is used to insert the contents of one PHP file into another PHP file.

If file not found → warning will appear but script continues.

---

# 🔹 require

require is similar to include but it stops the script if the file is missing.

If file not found → fatal error and script stops.
